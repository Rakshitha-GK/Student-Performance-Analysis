# Student-Performance-Analysis

##Project Overview
      This project analyzes student academic performance using Python.

##Tools Used
  - Python
  - Pandas
  - NumPy
  - Matplotlib
  - Seaborn

##Analysis Performed
  - GPA distribution
  - Study time vs GPA
  - Absences vs GPA

##Key Insights
  - Higher study time leads to better GPA
  - Increased absences reduce academic performance
  - GPA distribution shows most students fall in mid-range

##Dataset
  Kaggle - Student Performance Dataset
